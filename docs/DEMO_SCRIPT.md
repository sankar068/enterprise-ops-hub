# Demo Script

Follow this script to demonstrate the capabilities of InnoOps AI:

## 1. Authentication
1. Show the login screen.
2. Log in as an Employee using `employee@innoops.ai` / `demo123`.

## 2. IT Workflow (High Priority)
1. In the chat, type: `"My laptop camera is not working and I have an important meeting in one hour."`
2. Show that the AI automatically extracts urgency, assigns high priority, and recommends troubleshooting.
3. Confirm submission.

## 3. HR Workflow (Multi-turn)
1. Type: `"I want leave."`
2. Show the agent requesting follow-up dates and reasons.
3. Reply: `"Apply casual leave from June 20 to June 22 for personal work."`
4. Confirm submission.

## 4. Finance Workflow
1. Type: `"Submit ₹2,500 as travel reimbursement for a client meeting held on June 10. I have the receipt."`
2. Confirm submission.

## 5. Admin Dashboard
1. Log out. Log in as Admin using `admin@innoops.ai` / `demo123`.
2. Navigate to the `/admin` route.
3. Show live metric cards.
4. Filter by 'Pending' and click 'View Details' on the submitted IT ticket.
5. Add a timeline note and mark it as 'In Progress'.
